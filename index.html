📊 マジック / エリミネーションの定義
• 総試合数: NPBは各チーム143試合
• 勝率: 勝 / (勝 + 敗)（分は勝率に含めない）
• 優勝マジック: (2位の最大可能勝数) − (1位の現在勝数) + 1（0以下で✓）
• エリミネート(E): 1位勝数 ＞ チーム最大可能勝数, または (143+1) − 1位勝 − チーム敗 ≤ 0
• 記号: ✓＝確定, E＝不可能, -＝該当なし

🏆 プレーオフ (CS) 進出の判定
• 進出枠: 各リーグ上位3位がCS進出
• マジック: 4位の最大可能勝数を基準に、対象チームが到達すべき勝数（0以下で✓）
• エリミネート(E): (143+1) − 3位勝 − チーム敗 ≤ 0 または チーム最大可能勝数 ＜ 3位勝数
