📋 매트릭스 구조

  • 🟢 녹색: 0.5 초과 승률 (좋은 성과)
  • 🔴 빨간색: 0.5 미만 승률 (부진한 성과)
  • 🟡 노란색: 정확히 0.5 승률 (평균)

🎯 1위 탈환 가능성 분석 가이드

📊 기본 지표
  • 현재승수: 누적 승수
  • 게임차: 1위와 승수 차이
  • 잔여경기: 남은 경기 수
  • 최대승수: 전승시 최종 승수
🏆 1위 탈환 계산법
  • 1위팀 조건: 추격팀 전승시 1위가 져야할 경기수
  • 트래직넘버: 1위팀이 내 최대가능 승률을 넘기 위해 필요한 승수
  • 계산방식: 승률 비교 기준으로 1위 추월 가능성 분석
  • "불가능": 전승해도 1위팀 최저가능 승률을 넘을 수 없음
  • "역대기준": 87승 달성 필요 승률 및 가능 여부
📈 승률 난이도: ■ 0-45% ■ 45-55% ■ 55-70% ■ 70%+
⚾ 타이브레이커 표시: TB = 타이브레이커 필요 + = 상대전적 우위 - = 상대전적 열세 ⚾ = 타이브레이커 가능

📊 플레이오프 진출 조건 가이드

📈 지표 설명
  • 현재승수: 누적 승수
  • 잔여경기: 남은 경기 수
  • 최대승수: 전승시 최종 승수
🏟️ 플레이오프 진출 조건 계산법
  • PO 매직넘버: 5위팀(없으면 4위팀) 최대가능 승률을 넘어 PO 진출 확정하기 위해 필요한 승수
  • 계산방식: 승률 비교 기준으로 플레이오프 진출 확정점 분석
  • "자력 불가": 잔여경기 전승해도 5위팀 승률을 넘을 수 없음 (괄호 안 실제 필요 승수)
  • PO 트래직넘버: 나의 최대가능승수 - (5위팀 현재승수 - 1)
  • "탈락": 트래직넘버가 0 이하로 플레이오프 진출 불가능
🎯 역대 기준 매직/트래직넘버
  • 역대 기준 매직넘버: 72승까지 필요 승수
  • 역대 기준 트래직넘버: 71승 달성 불가능까지 남은 패수
  • 필요승률: 잔여경기 목표 승률
🏆 진출 색상: ■ 확정 ■ 유력 ■ 경합 ■ 탈락
📌 역대 기준: 매직넘버는 72승, 트래직넘버는 71승 기준으로 계산
